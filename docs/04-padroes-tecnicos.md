# 04 - Padrões Técnicos

## Stack definida (PRD)

- Python 3.13
- `google-api-python-client`
- `google-auth`, `google-auth-oauthlib`
- `pandas`, `openpyxl`
- `python-dotenv`
- `logging` (stdlib)

## Padrões de código

- PEP 8
- Uso de aspas simples
- Código, comentários e docstrings em inglês
- Execução via terminal (CLI)

## Padrões operacionais

- Logs com `RotatingFileHandler`
- Idempotência no processamento de e-mails
- Dependências mínimas necessárias
